# Daily-Rental-Analytics-Dashboard-Excel
This project is a daily rental management dashboard built in Microsoft Excel. It tracks tenant payments, overdue accounts, communication follow‑ups, and accountant updates. The dashboard is fully interactive, automated, and designed to showcase data analytics and visualization skills.

Dynamic Excel Table for structured data.

PivotTables to summarize:

Payment Status (On Track vs Overdue)

Communication Tracker (Phone Calls, Emails)

Accountant Updates (Pending vs Completed)
